# Sikulo Pizzeria

Projet compose d'un backend .NET et d'un frontend Angular.

## Commandes utiles

Depuis la racine du projet :

```powershell
npm start
```

Demarre le frontend Angular sur :

```text
http://localhost:4200
```

Pour compiler le frontend :

```powershell
npm run build
```

## Ou modifier le code ?

Modifie les fichiers sources ici :

```text
frontend/src
backend
database
```

Evite de modifier ces dossiers : ils sont generes automatiquement.

```text
frontend/dist
frontend/.angular
frontend/node_modules
backend/**/bin
backend/**/obj
```

Si tu vois une page blanche, ouvre d'abord :

```text
http://localhost:4200
```

N'ouvre pas directement les fichiers dans `frontend/dist`, car ce sont des fichiers compiles.
