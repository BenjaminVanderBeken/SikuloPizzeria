# Frontend Angular

Le code a modifier se trouve dans :

```text
frontend/src
```

Les dossiers suivants sont generes automatiquement :

```text
frontend/dist
frontend/.angular
frontend/node_modules
```

Ne modifie pas les fichiers dans `dist` : ils sont recréés a chaque build.

Depuis la racine du projet, utilise plutot :

```powershell
npm start
```

Puis ouvre :

```text
http://localhost:4200
```
